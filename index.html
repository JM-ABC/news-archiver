01 왜 만들었나
커머스 업무를 하다 보면 업계 뉴스를 매일 챙겨야 할 것 같은 압박이 있어요. 쿠팡은 오늘 뭘 발표했는지, Amazon은 어떻게 움직이는지, Temu는 또 치고 들어오는 것 같고… 그런데 막상 찾아보면 시간만 사라지고 정작 필요한 인사이트는 잘 안 남더라고요. 같은 기사를 반복해서 보기도 하고, 국내 뉴스에 집중하다 보면 글로벌 흐름은 자연스럽게 놓치기도 하고요. 이 문제를 자동화로 해결해보자 싶어서 만들게 됐습니다.
🕐
매일 뉴스 찾는 데만 1~2시간
여러 사이트를 돌아다니며 직접 검색하다 보면 정작 업무 시작은 늦어지고, 찾은 내용도 파편적으로 흩어져 있어요. 정보를 수집하는 게 목적이 아닌데 거기에 시간이 다 쓰이는 상황이었어요.
🔁
같은 기사가 계속 반복된다
하나의 이슈를 여러 매체가 동시에 다루다 보니, 실제로는 새로운 내용이 아닌데 새 기사처럼 계속 등장해요. 읽고 나서 "이거 어제도 봤는데?" 싶은 경험이 자주 있었어요.
🌐
글로벌 동향은 놓치기 쉽다
국내 뉴스 챙기기에도 바쁘다 보면 Amazon, Temu, TikTok Shop 같은 해외 커머스 흐름은 자연스럽게 후순위로 밀려요. 그런데 글로벌 트렌드가 결국 국내 시장에도 영향을 주더라고요.
02 어떻게 만들었나 — 파이프라인 7단계
1/7
RSS로 뉴스 수집
Google News 등 20개 피드에서 국내·글로벌 커머스 기사를 자동으로 가져옵니다.
RSS가 뭔가요? 뉴스 사이트들이 제공하는 "자동 구독 채널"이에요. 유튜브 채널을 구독하면 새 영상 알림이 오듯, RSS 주소를 등록해두면 새 기사가 올라올 때마다 자동으로 가져올 수 있어요.

feedparser는 이 RSS 데이터를 파이썬이 읽을 수 있는 형태로 변환해주는 도구예요. 코드 한 줄로 수백 개 기사를 한 번에 수집할 수 있어요.
feedparser
2/7
중복 기사 제거 (Deduplication)
최근 3일치 발행 내역과 비교해서 이미 다룬 기사는 자동으로 걸러냅니다.
Dedup(디덥)이 뭔가요? "중복 제거(Deduplication)"의 줄임말이에요. URL과 제목을 비교해서 이미 발행한 기사는 자동으로 빼줘요.
dedup · days=3
3/7
불필요한 기사 필터링 (Filtering)
광고성 홍보글, 인사발령 공시, 커머스와 관계없는 기사를 자동으로 제외합니다.
"체험단 모집", "리뷰 대행" 등 업계 분석에 필요 없는 기사들을 미리 정해둔 키워드 패턴으로 자동 제거해요.
filter
4/7
Claude AI로 기사 요약
각 기사를 소카테고리로 분류하고, 「현상 → 시사점」 2줄 구조로 요약합니다.
단순 요약이 아니라 사실과 의미를 한 세트로 정리해요. 기사를 클릭하지 않아도 핵심이 바로 잡혀요.
Claude Haiku 4.5
5/7
핵심 트렌드 3가지 도출
당일 수집된 기사 전체를 종합해 오늘 커머스 업계의 핵심 흐름 3가지를 자동으로 뽑아냅니다.
이메일 상단에서 바로 확인할 수 있어요. 바쁜 날엔 이것만 봐도 돼요.
Claude Haiku 4.5
6/7
Notion에 아카이브
날짜별로 Notion 페이지에 자동 업로드해서 나중에 검색하거나 찾아볼 수 있게 쌓아둡니다.
이메일은 흘러가지만 Notion엔 기록이 남아요. 아카이브 역할을 합니다.
Notion API
7/7
HTML 이메일로 발송
Outlook 호환 HTML 뉴스레터 형태로 렌더링해서 등록된 수신자에게 자동 발송합니다.
Resend 서비스를 사용해요. 월 3,000건까지 무료입니다.
Resend API
03핵심 차별점
1
기사 나열이 아닌 트렌드 해석
AI가 각 기사를 「▶ 현상 → 시사점」 구조로 해석해서 전달해요.
2
국내 + 글로벌 동시 뷰
국내외 플랫폼 소식을 한 이메일에서 나란히 볼 수 있어요.
3
중복 없이, 신선한 기사만
최근 3일치와 비교해서 이미 다룬 기사는 자동으로 빠져요.
