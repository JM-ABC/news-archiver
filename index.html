01 왜 만들었나
커머스 업무를 하다 보면 업계 뉴스를 매일 챙겨야 할 것 같은 압박이 있어요. 쿠팡은 오늘 뭘 발표했는지, Amazon은 어떻게 움직이는지, Temu는 또 치고 들어오는 것 같고… 그런데 막상 찾아보면 시간만 사라지고 정작 필요한 인사이트는 잘 안 남더라고요. 같은 기사를 반복해서 보기도 하고, 국내 뉴스에 집중하다 보면 글로벌 흐름은 자연스럽게 놓치기도 하고요. 이 문제를 자동화로 해결해보자 싶어서 만들게 됐습니다.
🕐
매일 뉴스 찾는 데만 1~2시간
여러 사이트를 돌아다니며 직접 검색하다 보면 정작 업무 시작은 늦어지고, 찾은 내용도 파편적으로 흩어져 있어요. 정보를 수집하는 게 목적이 아닌데 거기에 시간이 다 쓰이는 상황이었어요.
🔁
같은 기사가 계속 반복된다
하나의 이슈를 여러 매체가 동시에 다루다 보니, 실제로는 새로운 내용이 아닌데 새 기사처럼 계속 등장해요. 읽고 나서 "이거 어제도 봤는데?" 싶은 경험이 자주 있었어요.
🌐
글로벌 동향은 놓치기 쉽다
국내 뉴스 챙기기에도 바쁘다 보면 Amazon, Temu, TikTok Shop 같은 해외 커머스 흐름은 자연스럽게 후순위로 밀려요. 그런데 글로벌 트렌드가 결국 국내 시장에도 영향을 주더라고요.
02 어떻게 만들었나 — 파이프라인 7단계
1/7
RSS로 뉴스 수집
Google News 등 20개 피드에서 국내·글로벌 커머스 기사를 자동으로 가져옵니다.
RSS가 뭔가요? 뉴스 사이트들이 제공하는 "자동 구독 채널"이에요. 유튜브 채널을 구독하면 새 영상 알림이 오듯, RSS 주소를 등록해두면 새 기사가 올라올 때마다 자동으로 가져올 수 있어요.

feedparser는 이 RSS 데이터를 파이썬이 읽을 수 있는 형태로 변환해주는 도구예요. 코드 한 줄로 수백 개 기사를 한 번에 수집할 수 있어요.
feedparser
2/7
중복 기사 제거 (Deduplication)
최근 3일치 발행 내역과 비교해서 이미 다룬 기사는 자동으로 걸러냅니다.
Dedup(디덥)이 뭔가요? "중복 제거(Deduplication)"의 줄임말이에요. 같은 이슈를 10개 매체가 다루면 10개가 다 들어오는데, URL과 제목을 비교해서 이미 발행한 기사는 자동으로 빼줘요.

월·수·금 스케줄에서 가장 긴 간격이 금요일 → 월요일 (3일)이라, 이 기간에 겹치는 기사가 없으려면 최근 3일치를 비교해야 해요. days=3이 이 스케줄의 딱 맞는 값이에요.
dedup · days=3
3/7
불필요한 기사 필터링 (Filtering)
광고성 홍보글, 인사발령 공시, 커머스와 관계없는 기사를 자동으로 제외합니다.
Filter(필터)가 뭔가요? 커피를 내릴 때 원두 찌꺼기는 걸러내고 맑은 커피만 뽑아내는 필터와 같아요. "체험단 모집", "리뷰 대행", 임원 인사 공시처럼 업계 분석에 필요 없는 기사들을 미리 정해둔 키워드 패턴으로 자동 제거해요.
filter
4/7
Claude AI로 기사 요약
각 기사를 소카테고리로 분류하고, 「현상 → 시사점」 2줄 구조로 요약합니다.
단순 요약이 아니라 ▶ 쿠팡, 로켓배송 권역을 제주도로 확대 → 오프라인 유통과의 경쟁이 지방 물류로 전선 이동 처럼, 사실과 의미를 한 세트로 정리해요. 기사를 클릭하지 않아도 핵심이 바로 잡혀요.
Claude Haiku 4.5
5/7
핵심 트렌드 3가지 도출
당일 수집된 기사 전체를 종합해 오늘 커머스 업계의 핵심 흐름 3가지를 자동으로 뽑아냅니다.
개별 기사를 다 읽지 않아도 "오늘 커머스 업계에서 가장 중요한 움직임 3가지"를 이메일 상단에서 바로 확인할 수 있어요. 바쁜 날엔 이것만 봐도 돼요.
Claude Haiku 4.5
6/7
Notion에 아카이브
날짜별로 Notion 페이지에 자동 업로드해서 나중에 검색하거나 찾아볼 수 있게 쌓아둡니다.
이메일은 흘러가지만 Notion엔 기록이 남아요. "지난달 쿠팡 관련 기사 뭐가 있었지?" 할 때 바로 찾을 수 있는 아카이브 역할을 해요.
Notion API
7/7
HTML 이메일로 발송
Outlook 호환 HTML 뉴스레터 형태로 렌더링해서 등록된 수신자에게 자동 발송합니다.
Resend라는 이메일 발송 서비스를 사용해요. 월 3,000건까지 무료라 개인 뉴스레터 용도로는 비용이 들지 않아요.
Resend API

03 핵심 차별점
1
기사 나열이 아닌 트렌드 해석
링크 목록을 보내주는 게 아니라, AI가 각 기사를 「▶ 현상 → 시사점」 구조로 해석해서 전달해요. 수치/사실로 시작해서 업계 의미로 끝나는 포맷이라, 기사를 클릭하지 않아도 요점이 바로 잡혀요.
2
국내 + 글로벌 동시 뷰
쿠팡·무신사 같은 국내 플랫폼 소식과 Amazon·Temu·TikTok Shop 해외 동향을 한 이메일에서 나란히 볼 수 있어요. 글로벌 트렌드가 국내에 어떤 영향을 줄지 비교하면서 읽을 수 있어요.
3
중복 없이, 신선한 기사만
최근 3일치와 비교해서 이미 다룬 기사는 자동으로 빠져요. 발행할 때마다 진짜 새로운 소식만 담겨 있어요.
4
소카테고리로 분류
이커머스 전략, AI커머스, 라이브커머스, 물류/배송, 패션/뷰티 등 세부 분류로 자동 정리돼요. 오늘 내 관심 분야 기사만 먼저 읽는 것도 가능해요.
5
설정 이후엔 아무것도 안 해도 됨
GitHub Actions가 매주 월·수·금 오전 8시(KST)에 알아서 실행해요. 서버도 없고, 버튼도 안 눌러도 돼요. 일어나면 이메일이 와 있어요.
